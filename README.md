# Sensor de Proximidade -- IIOT

- Foi desenvolvido um circuito IIOT com o objetivo de um sensor de proximidade acender um led caso uma certa distancia estabelecida fosse atingida.
O projeto foi desenvolvido tanto web quanto fisico, o WEB foi desenvolvido pelo Wokwi e o fisico foi desenvolvido com materias físicos em sala de aula.
